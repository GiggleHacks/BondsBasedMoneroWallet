# Bond's Based Monero Wallet

A Monero desktop wallet for Windows. Your keys, your coins, your business.

[![Monero](https://img.shields.io/badge/Monero-XMR-FF6600?style=flat-square&logo=monero&logoColor=white)](https://getmonero.org)

Send, receive, and swap XMR. Real-time price chart. Node manager. Sound effects. Interactive 3D model in the sidebar.

Most importantly: **FILE YOUR TAXES.**

Watch your transactions upload over a Win95 dial-up modem animation and receive your official government compliance certificate.

## Download

Grab the portable `.exe` from [Releases](../../releases). No install needed. Windows 10/11.

## Build

```bash
npm install
npm run dev       # development
npm run package   # build exe
```
