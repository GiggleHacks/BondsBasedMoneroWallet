## вҡЎ What Is This?

**Bond's Based Monero Wallet** is a desktop application for Windows that lets you send, receive, and manage Monero (XMR) with zero compromise on privacy or user experience.

It was built from the ground up for people who actually understand why financial privacy matters вҖ” not as a talking point, but as a fundamental human right.

Under the hood it uses **monero-ts** (the official Monero JavaScript library) running inside Electron, giving you a true wallet with full cryptographic verification, not just a front-end to a centralized API.

## вңЁ Features

### рҹ’ё Wallet Core
- **Create & restore** wallets with 25-word Monero seed phrases
- **Multi-wallet support** вҖ” manage multiple wallets from one app
- **Send XMR** with full priority control (Low / Medium / High / Default)
- **Receive XMR** with QR code generation (standard `monero:` URI scheme)
- **Subaddresses** вҖ” create labeled receiving addresses for privacy
- **Sweep wallet** вҖ” send entire balance in one click
- **Live fee estimation** before broadcasting
- **USD вҶ” XMR** amount toggle on the send screen

### рҹ”„ Dual Sync Modes

| Mode | Speed | Privacy | How It Works |
|------|-------|---------|--------------|
| **Full Node** | Moderate | в–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳ Maximum | Your wallet talks directly to a Monero RPC node. Nobody sees your transactions. |
| **Light Wallet (LWS)** | вҡЎ Instant | в–Ҳв–Ҳв–Ҳв–Ҳв–‘в–‘в–‘в–‘ Good | Shares your *view key only* with a trusted LWS server. Spend key stays local. |

Switch between modes in Settings at any time.

### рҹ“Ҡ Price Tracker
- Real-time **XMR/USD price** via CryptoCompare (free API, no key required)
- Interactive **price chart** with 7 timeframes: 24H В· 1W В· 1M В· 3M В· 6M В· 1Y В· ALL
- Hover crosshair with price tooltip
- Period percentage change display
- Click the price badge on Dashboard to jump straight to the chart

### рҹҢҗ Node Management
- **6 default trusted nodes** pre-configured:
  - Seth For Privacy
  - HashVault
  - MoneroDevs
  - Monerujo
  - Stack Wallet
  - Cake Wallet
- **Latency testing** вҖ” test any node and see ms + block height
- **"Test All"** вҖ” benchmark all nodes simultaneously
- **Auto best-node selection** вҖ” picks the fastest healthy node on startup
- **Custom node** вҖ” point the wallet at your own node

### рҹ”Ҡ Sound System
Full **per-sound toggle** system with a master on/off switch:

| Event | Sound File |
|-------|-----------|
| App startup | `startup3.mp3` |
| Button click | `clicky2.mp3` |
| Money received рҹӨ‘ | `money.mp3` |

All sounds individually toggled in **Settings вҶ’ Sounds**.

### рҹ”’ Security
- **Password-protected wallets** вҖ” AES encryption via monero-ts
- **Seed phrase reveal** is gated behind password re-entry
- **View key isolation** вҖ” LWS mode never exposes your spend key
- **Context isolation** вҖ” Electron renderer runs in a secure sandbox
- **No Node.js in renderer** вҖ” strict Electron security model enforced

### рҹ“– Address Book
- Save contacts with names and Monero addresses
- Search, edit, delete contacts
- Cloud sync support for contacts across devices (via local folder)

### рҹ–ҘпёҸ Transaction History
- Full incoming and outgoing transaction list
- Search by transaction hash
- Filter by direction (in / out)
- Add **custom labels** to transactions for bookkeeping
- Confirmation status with block height

### рҹ“қ Logs & Diagnostics
- Real-time in-app log viewer (no log files to hunt)
- Filter by source: `wallet` В· `node` В· `lws` В· `sync` В· `app`
- Color-coded by severity level
- Max 500 log entries вҖ” rotating window, never bloats

- **Git**

```bash
# Clone the repository

cd based-monero-wallet

npm run dev
```

### Build Commands

```bash

npm run build
```

## рҹ”‘ First-Time Setup

1. **Launch the app** вҖ” the Matrix rain welcome screen appears
2. Choose **Create New Wallet** or **Restore from Seed**
3. **Set a strong password** вҖ” this encrypts your wallet file locally
4. The app auto-selects the **fastest available Monero node**
5. Your wallet begins syncing вҖ” progress shown in the sidebar
6. Once synced, you're ready to send and receive XMR

> рҹ’Ў **Tip:** For instant balance display, enable **Light Wallet Server** in Settings вҶ’ Light Wallet. Your view key is shared with the LWS server, but your spend key and seed never leave your machine.

**Electron Security:**
- Renderer process runs in a **sandboxed context** with no Node.js access
- All wallet operations happen in the **main process only**
- The preload script exposes a typed, minimal API surface via `contextBridge`
- No `eval()`, no `nodeIntegration: true`, no exceptions
